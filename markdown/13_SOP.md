# 13 — STANDARD OPERATING PROCEDURES

## SOP 1 — New Project

Trigger: Andrea announces a new project.

1. Cleo creates brief.
2. Scout creates project memory shell.
3. Cleo identifies owner agent(s).
4. Specialist produces first actionable plan.
5. Andrea approves next step if external/high-risk.

---

## SOP 2 — Multi-Agent Task

Trigger: request spans 2+ domains.

1. Cleo decomposes into sequence.
2. Scout gathers context if needed.
3. Specialist A produces input for Specialist B.
4. Cleo synthesizes.
5. Andrea receives final action pack.

Max 3 agents per cycle unless Andrea explicitly asks.

---

## SOP 3 — Agent Failure

Trigger: agent output is wrong, incomplete, or off-scope.

1. Cleo identifies failure class.
2. Route to fallback or correct specialist.
3. Scout logs if material/repeated.
4. Update binding/profile only if pattern emerges.

---

## SOP 4 — Memory Conflict

Trigger: stored memory contradicts Andrea or current project.

1. Scout presents Stored vs Current.
2. Andrea confirms.
3. Scout updates memory.
4. Conflict logged if material.

---

## SOP 5 — Playbook Update

Trigger: Andrea requests change or review identifies pattern.

1. Draft exact file edits.
2. Check consistency with 00, 01, B, C.
3. Remove obsolete contradictions.
4. Andrea approves material changes.
5. Version bump.

---

## SOP 6 — Code Change

Trigger: code/script/backend request.

1. Cleo routes to Max or Codex.
2. Agent checks scope/risk.
3. Non-destructive implementation.
4. Verification.
5. Delivery with run instructions.
6. Production/deploy waits for Andrea.

---

## SOP 7 — UI/Webapp Work

Trigger: UI/frontend/design request.

1. Pixel defines UX goal.
2. Max involved only if backend/API needed.
3. Component/prototype built.
4. Verification on target viewport.
5. Andrea reviews before public deploy.

---

## SOP 8 — Legal/Compliance

Trigger: contract, policy, legal, procurement risk.

1. Route Lex/Counsel.
2. Identify obligations and red flags.
3. Severity-rate issues.
4. Suggest negotiation/action.
5. No final legal decision without qualified professional when binding.

---

## SOP 9 — Visa/Immigration

Trigger: visa, passport, permits, immigration, embassy.

1. Visa agent identifies jurisdiction and objective.
2. Use current official-source dependency.
3. Build checklist and timeline.
4. Flag unknowns.
5. Andrea confirms before applications/submission.

---

## SOP 10 — Weekly Review

1. Cleo gathers status.
2. Scout supplies project/memory deltas.
3. Specialists provide blockers.
4. Cleo outputs completed / in progress / next / risks.
